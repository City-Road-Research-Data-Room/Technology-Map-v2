Tier 01 · Lean
Launch & validation
≈ $18k / month
  • Single-region cloud compute (managed EC2 / VPS class)
  • Dev-tier odds feed ($30–$500/mo class) + free Polymarket / Kalshi REST & WebSocket
  • Shared Polygon RPC; ClickHouse + Postgres on managed instances
  • REST polling + WebSocket; reconciliation over public endpoints
Latency target — seconds. Event-driven, not a speed race.
Tier 02 · Competitive
Scaled deployment
≈ $40–58k / month
  • Multi-AZ reserved compute; isolated execution + research planes
  • Mid / streaming odds + sharp-reference feeds (SSE / WebSocket)
  • Kalshi FIX 4.4 session; dedicated Polygon archive RPC + indexer
  • Polymarket RTDS market-maker feed; redundant WebSocket sessions
Latency target — sub-second on WebSocket; FIX for Kalshi order flow.
Tier 03 · Top-tier
Mature capacity
≈ $70–85k / month
  • Single-tenant bare-metal (.metal) where it helps; PTP time-sync
  • Enterprise odds / sharp data ($3k–$10k+/mo class, multi-sport)
  • Dedicated RPC + redundancy; full replay & markout analytics
  • Capacity reservations; venue-floor execution where it adds edge
Latency target — venue-floor. On-chain settlement caps the real benefit.
